# Tasteful Hoarders Tracking App

A CLI Python app utilizing microservices to create an experience allowing users to keep track of their tasteful collections.

Contained are the following microservices:
* report microservice
* ascii microservice
* currency exchange microservice
* name generator microservice

Through development, I gained skills through various experiences such as:
* the code required to process an image into an ascii version
* using distinct ports in order to run the required microservices in different processes
* to decide on the communication pipeline that will allow microservices to programmatically communicate with the main program
* how the microservice structure does not utilize direct function calls or user interaction
* how developing an app with user stories and inclusivity heuristics creates a user friendly product with features that would fulfill user needs
* the difference between functional and non-functional requirements, and how that impacts design choices
* how a task management software can keep momentum in development that encourages feature completion in a timely manner

I have future plans to add the following features:
* track by uploading images: incorporate image classification training to automatically recognize an item of a collection
